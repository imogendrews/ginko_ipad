# Ginko Treeee

This project is a **React + Vite + shadcn/ui** app using **Tailwind CSS**.  


---

## Features

- Sequential text areas for visitor input
- The second question is disabled until the first is submitted
- After the second submission, the first question updates dynamically
- Dark-mode UI optimized for exhibition screens
- Built entirely with **shadcn/ui** components (Textarea, Button)
- Easily extensible â€” add more questions or components with shadcn

---

## Prerequisites

- Node.js >= 18
- npm (or yarn/pnpm)
- Terminal or code editor

## Getting Started

1. **Clone the repository**

```bash
git clone https://github.com/imogendrews/ginko_ipad.git
cd your-repo
```

2. **Insall libs**
npm install

3. **Run the app locally**

npm run dev

4. **Build for Production**
npm run build
